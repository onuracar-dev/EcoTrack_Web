# EcoTrack - Sustainable Living & Waste Tracking Platform

EcoTrack is a community-focused web platform designed to promote environmental awareness by digitizing individual recycling processes. This project is the vanilla web implementation of the EcoTrack ecosystem.

## 🚀 Features
* **Impact Dashboard:** Instantly calculates the environmental benefits (saved water, trees, and energy) of recycled waste using vanilla JavaScript logic.
* **Waste Guide (Search Alerter):** A quick-search directory helping users categorize their waste correctly.
* **Interactive Recycling Map:** Localizes nearest recycling hubs and drop-off points based on categories.
* **Lightweight & Responsive:** Native HTML5 and CSS3 architecture ensuring fluid performance across all devices without heavy frameworks.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Data Storage:** LocalStorage (for client-side persistence of waste history)

# EcoTrack - Sürdürülebilir Yaşam ve Atık Takip Platformu

EcoTrack, bireysel geri dönüşüm süreçlerini dijitalleştirerek çevre bilincini artırmayı hedefleyen toplum yararına bir web platformudur. Bu proje, EcoTrack ekosisteminin saf (vanilla) web uygulaması sürümüdür.

## 🚀 Özellikler
* **Etki Paneli (Dashboard):** Geri dönüştürülen atıkların doğaya sağladığı somut faydaları (tasarruf edilen su, kurtarılan ağaç ve enerji) saf JavaScript mantığıyla anlık olarak hesaplar.
* **Atık Rehberi:** Kullanıcıların ellerindeki atıkları doğru şekilde kategorize etmelerine yardımcı olan hızlı arama dizini.
* **İnteraktif Geri Dönüşüm Haritası:** Atık türlerine göre en yakın toplama merkezlerini ve bırakma noktalarını listeler.
* **Hafif ve Esnek (Responsive) Tasarım:** Büyük framework'ler kullanılmadan, tüm cihazlarda akıcı performans sağlayan yerel HTML5 ve CSS3 mimarisi.

## 🛠️ Kullanılan Teknolojiler
* **Önyüz (Frontend):** HTML5, CSS3, Saf JavaScript (Vanilla JS)
* **Veri Saklama:** LocalStorage (Atık geçmişinin kullanıcı tarayıcısında kalıcı olarak saklanması için)
